About:

MyUniTracker is an easy to use tool designed to give university students an overview of their units semester by semester. It gives them a general overview of how they are going in all their units, as well as giving useful statistics on their GPA (grade point average) and WAM (weighted average mark). It also has a breakdown of each individual unit, by allowing students to add assessments as they get the marks back, and to receive useful statistics such as their current percentage and grade. At the end of semester it will show students what percent they need in the final exam to achieve a selected grade.


Features:

The application facilitates the common requirements a university student wants to be able to see but at the press of the button, instead of tediously figuring out these measures. The key features of the application include:

  • Adding, editing and removing units.
  • Adding, editing and removing assessments completed (or even future assessments, by predicting your mark).
  • Viewing your progress in a specific unit via a graph of assessment results.
  • Gives a combined overview of all your results and presents it in just one graph, and also shows your WAM and GPA (see next point).
  • Adding past results so the application can calculate your current GPA and WAM (following how UWA does it), and predicts the future values for these measures based on your results for the currently listed units, at that point in time.
  • Allows users to view their Major/s WAM and expected WAM, as this can be an important way to measure success within a specific degree.
  • Managing which units are displayed in the Overview panel.

Installation:

Please ensure that you have at a minimum, Java Runtime Environment 6 (anything above 6 is perfectly fine too). To check your java version, open Terminal (Mac) or Command Prompt (Windows) and type in java -version then press enter.

Windows:

You will need the most recent version of the MyUniTracker app installer. When installing do NOT install to Program Files, install either in Documents or on the C: drive. See the Guide for detailed installation instructions. For Java I recommend either
  • Java Development Kit

    • or

  • Java Runtime Environment

Mac OSX

You will need the most recent version of the MyUniTracker app. See the Guide for detailed installation instructions. For Java I recommend the Java Development Kit.
